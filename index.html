About Code Weaver's Quick-Cache

This is a secure, temporary online notepad for code snippets and text, powered by Pastebin. Features include syntax highlighting, expiration, privacy controls, and more. Your notes are never stored on this site, only on Pastebin via a secure proxy.

  • Syntax Highlighting: Choose your language for code preview and Pastebin formatting.
  • Expiration: Set how long your note should last on Pastebin.
  • Privacy: Unlisted (only those with the link can view) or Public (searchable on Pastebin).
  • Copy & Vanish: Copies the link and clears your note for privacy.

For feedback or issues, contact the developer.
